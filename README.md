# PanelDeControl

Modo frontend-only: esta rama/versión muestra una maqueta del panel de enlaces sin conexión a base de datos. Se usan enlaces en memoria (hard-coded) para ver el diseño.

Cómo ejecutar:

- dotnet restore
- dotnet run
- Abrir http://localhost:5000

Notas:
- Las páginas Create/Edit/Delete siguen en el proyecto pero en esta versión no están conectadas a ninguna base de datos.
- Si quieres que cambie los enlaces de ejemplo por los tuyos, dímelos y los actualizo en Pages/Index.cshtml.cs.
